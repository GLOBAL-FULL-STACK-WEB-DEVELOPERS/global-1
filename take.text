Ma kala taqaana git iyo GitHub?

Marka aad ku jirto technology waxa laga marman ah inad aqoon u leedahay git iyo github Hadan sii fudaydiyo git waa tool aad ku maamulayso code kaga kaas oo diwaan galin kaydinayana isbadal walba oo ku imaada projectgaga.

Tusaale hadaan sameeyo file cusub uu aqoonsanaya markaba, hadii aan code ku qoro file kas cusub iyana uu diwaangalin dhawaan hadan wax ka badalo uu track garayn hadan dib ka dareemo in code kaygu khaldanyahay uu ii fudaydin in aan dib ugu noqdo oo aan xaliyo code kayga sido kale waxa uu ka caawinaya in projectgas koox ahaan looga shaqeeyo ama si iskaashi ah loo dhiso iyada oo qofwalba fahmayo cida code ka wax ka badashey iyo cida wax kuso darteyba.

Halka github na uu yahay shabakad ku ogolanaysa inad code kaga la wadaagto developers ka kale,sido kale waa sanduuq aad ku kaydsanayso code kaga sido kale waxa uu inaga caawinaya inan wax iska barano tusaale hadan github account leeyahay waxan iman kaaga aniga oo akhrisan codes kaga iyo sida aad projects walba u dhisto taas oon ka heli khibrad adiguna sido kale projects kayga sida aan u dhisey wad so akhrisan kartaa sido kale waxa uu ka caawinaya inad khibrada iyo tayadadaba so bandhigtid hadad rabto inad shaqo hesho ama aad project aad dhistey gadaysid github waa meesha ugu munaasibsan eed ku so bandhigi kartid si koobana waxan odhan karnaa waa baro bulsho gaar u ah developers ama programmers ka.
